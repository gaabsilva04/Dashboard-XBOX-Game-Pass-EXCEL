# Dashboard de Análise de Vendas - Xbox Game Pass

## 📊 Visão Geral

Este dashboard apresenta uma análise abrangente das vendas do serviço **Xbox Game Pass**, fornecendo insights detalhados sobre o desempenho dos diferentes planos e modalidades de assinatura. A ferramenta permite acompanhamento em tempo real e comparação de períodos para tomada de decisão estratégica.

## 🎮 Planos Analisados

### 1. **EA Play Season Pass**
- Acesso exclusivo aos títulos da Electronic Arts (EA)
- Conteúdo sazonal atualizado regularmente
- Integração com o catálogo principal do Game Pass

### 2. **Minecraft Season Pass**
- Acesso a conteúdo exclusivo de Minecraft
- Atualizações de temporada e itens especiais
- Complemento premium para jogadores de Minecraft

### 3. **Total de Vendas**
- Consolidação de todas as modalidades de assinatura
- Visão holística do desempenho geral do Game Pass
- Métricas de crescimento e tendências

## 📅 Filtros de Análise

O dashboard oferece três períodos de análise para flexibilidade na visualização de dados:

- **Análise Mensal**: Acompanhamento detalhado de variações mês a mês
- **Análise por Quadrimestre**: Visão trimestral para identificar tendências sazonais
- **Análise Anual**: Perspectiva de longo prazo com comparação ano a ano

## 📈 Principais Métricas

- **Volume de Vendas**: Quantidade de assinaturas ativas por plano
- **Receita Gerada**: Faturamento consolidado por período
- **Taxa de Crescimento**: Evolução percentual entre períodos
- **Distribuição de Planos**: Proporção de vendas por categoria
- **Sazonalidade**: Variações esperadas em períodos específicos

## 🎯 Objetivo

Fornecer uma visão clara e objetiva do desempenho do Xbox Game Pass, facilitando:
- Análise estratégica de vendas
- Identificação de padrões e tendências
- Comparações entre períodos
- Suporte à tomada de decisão gerencial

## 📝 Notas Importantes

- Os dados são organizados por período (mês, quadrimestre, anual) para máxima flexibilidade
- O dashboard integra informações de múltiplas fontes de dados
- Atualizações regulares garantem informações precisas e confiáveis
